# CellAutoDesktop — Клеточные автоматы на Rust

> **Автор:** Катаржин Григорий, Алексеев Григорий  
> **Язык:** Rust  
> **Платформа:** Desktop 
> **Группа:** ИСП-232

---

## Что это?

**CellAutoDesktop** — это нативное десктопное приложение для симуляции клеточных автоматов, реализованное на языке программирования **Rust**. Проект ориентирован на высокую производительность, работу с большими объёмами данных и динамически изменяющимся миром.

 Мир представлен как логически бесконечная двумерная сетка, разбитая на чанки (64x64 клетки), что обеспечивает эффективное хранение только активных областей.

---

## Что реализовано?

### Функциональность
- ▶️/⏸️ Запуск, пауза и остановка симуляции
- ⏭️ Пошаговое выполнение (Step)
- 🎚️ Управление скоростью симуляции (TPS: 1–60 тиков/сек)
- 🔍 Масштабирование и перемещение области просмотра (drag + колесо мыши)
- 🔄 Сброс мира к выбранному пресету
-  20+ встроенных пресетов
- ⌨️ Горячие клавиши (Space, R, N, H, стрелки, +/-)

### Архитектура
- Логика симуляции изолирована от UI (работает в отдельном потоке)
- Мир хранится разреженно через `HashMap<u64, Chunk>` с использованием Morton-кодирования координат чанков
- Чанки размером 64x64 клеток, каждая клетка — бит в `u64` → максимальная эффективность по памяти и скорости
- Параллельный расчёт следующего состояния через `rayon`
- Динамическое изменение числа активных элементов — мир растёт и сжимается автоматически

###  Интерфейс
- Панель управления слева (кнопки, слайдеры, список пресетов)
- Центральная область — визуализация мира
- Верхняя панель с названием и кнопкой помощи
- Поддержка горячих клавиш и подсказок

## Технологии

- **Rust 2024 Edition**
- [`eframe`](https://github.com/emilk/egui/tree/master/crates/eframe) — фреймворк для создания GUI-приложений
- [`egui`](https://github.com/emilk/egui) — immediate mode GUI
- [`rayon`](https://github.com/rayon-rs/rayon) — параллельные вычисления 
- `std::sync::{Arc, Mutex, RwLock}` — безопасное многопоточное взаимодействие
- `std::collections::{HashMap, HashSet}` — разреженное хранение мира

## Горячие клавиши

| Клавиша | Действие                  |
|---------|---------------------------|
| `Space` | Старт / Пауза             |
| `R`     | Сброс к текущему пресету  |
| `N`     | Следующий шаг             |
| `H`     | Скрыть/показать справку   |
| `↑↓←→`  | Перемещение вида          |
| `+/-`   | Увеличить/уменьшить зум   |
| Drag    | Перемещение мышью         |
| Scroll  | Масштабирование           |
